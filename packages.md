## custom-title

タイトルバーにファイルのパス他を表示する

```js
<% if (projectPath) { %>
	<%= projectName %>
	 <% if (relativeFilePath) { %>
	- <%= relativeFilePath %>
	<% } else { %>
	- <%= fileName %>
	<% } %>
<% } else { %>
	<%= filePath %>
<% } %>
<% if (gitHead) { %> [@<%= gitHead %>]<% } %>
```
- project管理下ならプロジェクト名と相対パス表示
- git管理下ならheadブランチ表示
